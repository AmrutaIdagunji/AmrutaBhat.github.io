Projects

Anomaly Detection

with Variational Auto Encoders

This project utilizes Variational Autoencoders (VAEs) to perform anomaly detection, with a specific focus on identifying deviations from normal operational behavior in hard disks. The dataset, obtained from Kaggle, consists of daily snapshots for each functioning hard drive, including a "failure" column with values of 0 and 1. Here, 1 indicates the last time point when the hard disk was active before a potential failure.

  • PyTorch
  • Torchvision
  • sklearn

Sentiment Analysis

with RNN and LSTM

This project does sentiment analysis on Airline tweets dataset. Annotated data helps Neural Network Models classify the tweets into Positive, Negative or Neutral. In the data preprocessing stage, class imbalances were handled, emojis, special characters were excluded and then word tokenized for modelling. Different LSTM model Architectures (Vanilla, Bi-directional, stacked) and GRU were trained to achieve best results.

  • NLTK
  • PyTorch
  • sklearn

Sporting Object Vision

with CNN and YOLO

This project focuses on developing an advanced object detection system specifically tailored for tennis. Utilizing a combination of state-of-the-art models including TrackNet, YOLO (You Only Look Once), and OpenCV, the system aims to accurately track tennis balls during matches. This provides a valuable tool for players and coaches, offering detailed game analysis and insights that were previously available only with high-end, expensive professional sports technologies.

  • Python
  • Opencv
  • PyTorch
